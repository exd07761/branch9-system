Backup & Restore

Manual backups, disaster recovery, and migration between Firebase projects. Administrator only.

Create Backup

Downloads a single JSON file containing every hearing, case, activity log entry, and user account currently in the system.

Restore from Backup

Select a previously downloaded backup file. Existing records are updated, missing records are created. Nothing already in the system is ever deleted.
